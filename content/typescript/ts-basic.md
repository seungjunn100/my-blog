## TypeScript란?

TypeScript는 JavaScript에 타입을 추가한 언어이다.

## 기본 타입

```ts
let name: string = "Junn";
let age: number = 30;
let isActive: boolean = true;
```

## 장점

- 타입 안정성
- 코드 가독성 향상
- 유지보수 용이

## 정리

TypeScript는 규모가 커질수록 더 큰 장점을 가진다.
